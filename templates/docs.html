Shop Savvy Documentation

Introduction

Shop Savvy is a unique online shopping platform that provides an enhanced user experience by allowing image-based product searches. The application is built with the Flask web framework and integrates a custom-trained machine learning model with Google Search.

Functions

upload_image(image_path: str) -> str

This function uploads an image to an image hosting service using the command-line utility "images-upload-cli". It returns the URL of the uploaded image, or None if an error occurred.

upload_file()

This function handles the image upload and search functionality. It validates the image, uploads it using the upload_image function, performs a Google Lens search with the image URL and the provided query string, and displays the search results to the user.

register()

This function handles the user registration process. It takes the user input, validates it, hashes the password, and stores the new user in the database.

login()

This function handles the user login process. It checks if the entered credentials match with any user in the database, and if so, logs the user in and redirects them to the homepage.

logout()

This function logs the user out by removing their username from the session and redirects them to the login page.

Pages

Home / Index Page

The home page is where users can upload an image and enter a search query to find similar products online.

Login Page

The login page allows users to enter their credentials and log in to their account.

Registration Page

The registration page allows users to create a new account by providing their username and password.

Profile Page

The profile page displays the user's username and allows them to log out.

Results Page

The results page displays the search results with similar or identical items available for purchase online.

Workflow

  • The user begins by registering for an account on the registration page.
  • After successful registration, they are redirected to the login page.
  • On the login page, they can enter their credentials to log into their account.
  • Once logged in, the user is taken to the home page.
  • On the home page, the user can upload an image and enter a search query.
  • The application then performs a Google Lens search with the uploaded image and the query string.
  • The search results are displayed on the results page, showing similar or identical items available for purchase online.
